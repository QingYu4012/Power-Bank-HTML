# Power Bank 串口调试助手

基于 Web Serial API 的网页版充电宝数据监测与串口透传调试工具，开箱即用、单页部署。

## 2026-04-17 功能实现记录 
1. 新增完整单页应用 [index.html](./index.html)。
2. 实现串口授权、端口列表刷新、连接/断开和热插拔监听。
3. 支持串口参数配置：波特率（自定义）、数据位、停止位、校验位、流控。
4. 支持字符串/HEX 双发送模式，支持自动换行（无/LF/CRLF/CR）。
5. 实现实时接收显示，支持字符串/HEX/字符串+HEX 三种接收视图。
6. 支持日志功能：实时日志、自动滚动、清空日志、保存日志到本地文件。
7. 支持定时循环发送：周期可配置，支持启动/停止，带防重入机制。
8. 支持文件发送：按分片发送、可设置分片大小与间隔、可取消并展示进度。
9. 完成响应式界面优化，适配桌面端和移动端布局。
10. 增加异常处理和状态反馈，优化连接失败、写入异常、设备断开等场景体验。

## 功能清单

- 暗黑主题主监测界面（SOC/SOH、电池信息、端口信息、异常占位）
- USB 权限检测、连接/断开、热插拔监听与自动重连尝试
- 串口配置弹窗（端口选择、常用波特率、自定义波特率）
- RTC 读取与同步
- 协议通信（`AA BB` 帧头、`CRC8-CCITT` 校验、帧缓冲解析）
- 自动轮询（电池 1s、端口 400ms）
- 开发者模式（5 连击进入）
- 透传发送（文本/HEX、行尾附加、定时发送）
- 透传日志和收发字节统计
- USB 命令面板（GET_ACK/GET_INFO/GET_DATA/GET_RTC/SET_RTC/GET_PORT）
- 管理员面板（GET_TEST/SET_TEST）
- 响应式布局与安全上下文提示（HTTPS/localhost）

## 使用方法

1. 使用 Chrome 或 Edge（Chromium 内核）打开页面。
2. 点击“连接”，按浏览器弹窗完成 USB 授权；或先点状态按钮进入“虚拟串口设置”选择端口和波特率。
3. 连接成功后在主界面查看实时数据：SOC/SOH、电池信息、端口信息。
4. 如需校时，点击“同步RTC”。
5. 连续点击顶部标题 5 次进入“开发者模式”。
6. 在开发者模式可进行文本/HEX 透传发送、定时发送、查看日志与字节统计。
7. 点击“命令”进入 USB 命令面板，按需发送 GET_DATA / GET_RTC / GET_PORT 等命令。
8. 如需测试命令，先“管理员登录”后使用 GET_TEST / SET_TEST。
9. 调试结束点击“断开”。
