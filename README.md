# Power Bank 串口调试助手

基于 Web Serial API 的网页版串口调试工具，面向串口透传调试场景，开箱即用、单页部署。

## 使用方法

1. 使用 Chrome 或 Edge（Chromium 内核）打开页面。
2. 点击“选择串口”进行设备授权。
3. 选择端口并设置波特率等参数后点击“连接”。
4. 在发送区域输入内容，选择发送模式后点击“发送”。
5. 如需自动发送，设置循环周期后点击“启动循环发送”。
6. 如需发送文件，选择文件并设置分片参数后点击“开始发送文件”。
7. 可随时清空日志或导出日志文本文件。

## 部署说明

- 本项目为纯静态页面，无需后端服务。
- 入口文件为 [index.html](./index.html)。

  - `https://` 域名
  - `http://localhost`
